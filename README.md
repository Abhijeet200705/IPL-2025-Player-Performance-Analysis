# IPL 2025 Data Analysis & Power BI Dashboard

## 📌 Project Overview

This project focuses on Exploratory Data Analysis (EDA) and Power BI Dashboard development using the IPL 2025 dataset. The objective is to uncover meaningful insights from batting statistics, perform predictive analysis steps, and create an interactive dashboard for visual storytelling.

## 📂 Dataset

**Source:** Kaggle IPL 2025 Dataset

Dataset Link:
https://www.kaggle.com/datasets/vinayaks0n1/ipl-2025

Files Used:
- IPL2025Batters.csv
- IPL2025Bowlers.csv
- IPL.ipynb
- IPL2025_Dashboard.pbix

---

## 🎯 Project Objectives

- Perform comprehensive Exploratory Data Analysis (EDA)
- Clean and preprocess the dataset
- Identify trends and patterns in player performance
- Conduct predictive analysis workflow
- Create at least 10 visualizations in Jupyter Notebook
- Build an interactive Power BI dashboard

## 🚀 How to Run

### Clone Repository

```bash
git clone <repository-link>
```

### Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

### Run Notebook

```bash
jupyter notebook IPL.ipynb
```

### Open Dashboard

Open `IPL2025_Dashboard.pbix` using Power BI Desktop.
